# 🚀 SpaceX Launch Success Prediction

An end-to-end Data Science and Machine Learning project focused on analyzing and predicting SpaceX rocket launch outcomes using real launch data.

This project includes:
- Data Collection & Cleaning
- Exploratory Data Analysis (EDA)
- Interactive Visualizations
- Geospatial Mapping
- Feature Engineering
- Machine Learning Classification Models
- Hyperparameter Tuning
- Dashboard Development using Plotly Dash

---

# 📌 Project Overview

The objective of this project is to predict whether a SpaceX rocket launch will successfully land based on launch parameters such as:

- Payload Mass
- Orbit Type
- Booster Version
- Launch Site
- Flight Number
- Landing Pad
- Reused Booster Information
- Geographic Coordinates

The project compares multiple machine learning algorithms and identifies the best-performing classification model.
